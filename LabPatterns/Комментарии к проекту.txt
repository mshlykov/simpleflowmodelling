1. Проект RandNums:

Он отвечает за генерацию выборок. При старте необходимо ввести объемы двух обучающих выборок,
а также объемы тестовых выборок. Программа сгенерирует их следующим образом: все выборки - это вектора 
с двухмерным нормальным распределением, причем представители разных классов разделены по разным файлам.
Представители первого класса смещены на вектор (2, 2), а второго - на вектор (-2, -2). Далее мы будем 
считать, что на представителях первого класса разделяющая функция положительна, а второго - отрицательна.

Обучаюшие выборки:
Teaching1.txt - представители первого класса
Teaching2.txt - представители второго класса

Тестовые выборки:
Test1.txt - представители первого класса
Test2.txt - представители второго класса

2. Проект Proj1:

Проект, в котором реализован сам метод. Применялось обучение на основе метода потенциальных функций. Поскольку
значения разделяющей функции для обучающих выборок изначально не заданы, обучение проводится расстановкой
коэффициентов 0, 1 или -1. В качестве потенциальной функции выбрана 
U(x, y) = 1.0 / (1.0 + 10 * d(x,y))

В консоль выводятся тестовые выборки, а также процент ошибок при их распознании.

На экран выводятся:

Красный цвет - обучающая выборка из представителей первого класса
Зеленый цвет - обучающая выборка из представителей второго класса

Желтый цвет - тестовая выборка из представителей первого класса
Зеленый цвет - тестовая выборка из представителей второго класса

Белым цветом приблизительно отображается уравниение Phi(x, y) = 0, где
Phi - разделяющая функция. Визуализация происходит из условия abs(Phi(x, y)) < 0.0001 проходом по сетке
экрана с шагом 0.01.

Данный проект лучше запускать в режиме Release.
